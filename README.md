# CTD Todo List

Code The Dream Todo List App built with React + Vite.

## Getting Started

### Installation
1. Clone the repository:
```bash
git clone https://github.com/EmmanuelCobian/ctd-todo-list.git
cd ctd-todo-list
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
